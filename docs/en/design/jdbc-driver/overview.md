# JDBC Driver Overview

JustDB JDBC driver provides standard JDBC interfaces, allowing Java applications to use JustDB in a declarative way.

## Core Features

- **Standard JDBC interface**: Implement complete JDBC specification
- **In-memory database**: Support in-memory mode, no real database required
- **SQL queries**: Support SELECT, INSERT, UPDATE, DELETE
- **Transaction management**: Complete transaction support (ACID)
- **Metadata queries**: DatabaseMetaData interface
